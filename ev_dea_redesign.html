Step 2 — Combined CEM (RC_ij)
RC_ij = d⁻_ij / (d⁺_ij + d⁻_ij) โดย d⁺ = (1−θ_min)+(1−θ_max), d⁻ = θ_min + θ_max
ค่าสูง (→ 1.0) = รุ่น i มีประสิทธิภาพสูงในมุมมองของรุ่น j | เส้นทแยง (bold) = self-evaluation
Step 3 — MSDT Weights (ω_j)
σ_j = population std dev ของ column j → ω_j = (Σσ − σ_j) / [(n−1)·Σσ]
Step 4 — การจัดอันดับ
w_i = Σ_j ω_j · RC_ij (Eq. 16) → w_i มากกว่า = ประสิทธิภาพรวมดีกว่า
